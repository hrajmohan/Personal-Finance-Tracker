# Personal-Finance-Tracker
This is a Personal Finance Tracker application written in Java. 

In this application, you can open existing files that hold transactions or add a new file holding transactions you want to the record. 
![Opening Existing File or Adding New File](pictures\adding_new_transaction_entry.png)

Each file designed to record transactions for each recorded month (MM-YYYY format). 
![B](pictures\month_and_year.png)


The 4 transaction categories covered here are Home, Food, Transportation, Entertainment. 

For each transaction entry you enter the category, transaction name, cost, payment type, additional details (store name, subscription, etc). 
![A](pictures\adding_new_transaction_entry.png)

As more entries are added to a monthly transaction file: you can see the total amount spent, total amount of transactions, and average cost per transaction. 
![A](pictures\file_entries.png)
